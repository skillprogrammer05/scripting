# scripting
it is a file that consist of set of commands that will execute in a frequent order in order to make repeated tasks to run automatically instead of running manually everytime.
# what is shebang?
it tells the operating system which interpretor should execute the file.
Every script starts with the shebang i.e **#!/bin/bash**
